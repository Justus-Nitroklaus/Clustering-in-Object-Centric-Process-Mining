User Manual


{% load static %}

  • Click on the Choose Files button (1) and select an OCEL from your computer.
  • Press the „Import Log“ button (2), if the upload of your file(s) was successful you will see it at (3).
  • Then press the „Clustering DFG page“ button (4) to get to the draw page.
  • You can also delete files by clicking on them and then pressing the „Delete“ button above button (4).


{% load static %}

  • Select an OCEL from the ones you uploaded before.
    In case you uploaded multiple ones with the same name, they will automatically get a suffix like the second and third OCEL file in the picture.


{% load static %}

  • Then you get to select the object type you want to cluster for.
    There will be listed all the possible object types from YOUR OCEL, which are therefore not necessarily the same as the ones shown in the example picture.


{% load static %}

  • Now you can select the attributes of the chosen object you want to cluster for.
    You can select one or multiple attributes.
    If you select more than one the average between these will be used to calculate the result.


{% load static %}

  • After that, you can also set filters.
    You can set the minimum number of activities (1) and the minimal number of edges (2), both in a range of zero to 5000.
    After you change the number you need to press the „Filter“ button (3).
  • The last step ist to select a clustering method (4) and the method how to assign events to the clusters (5).
    (Explanation: If you choose „All“ an event is just assigned to a cluster if ALL objects in that event are in that cluster.
    If you choose „Existence“ an event is assigned to a cluster as soon as AT LEAST ONE object from the event is in the cluster).
  • Then press the „Draw“ button (6).
